InitFetchStrategy

InitFetchStrategy

Defined in src\components\Autocomplete.vue:164

Enum {none, onInit, onFocus}

Defines the time of initial load the item list.

  • onInit: means the item list will be loaded on component is created.
  • onFocus: means the item list will be loaded on component is first focused. (By default)
  • none: means the item list will be loaded on user inputs first character.

Default: onFocus

serverSearch

Boolean

Defined in src\components\Autocomplete.vue:137

Flag (true|false).

TRUE: If it is true all changes of user input will be send to the remote server and the response data override the existed item list. It means the server side filtering.

FALSE: If it is false all changes of user input will be applied on the locale item list. It means local filtering.

Default: false
